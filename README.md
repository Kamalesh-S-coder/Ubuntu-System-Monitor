# 🖥️ Ubuntu System Monitor

A lightweight, terminal-based system and network monitor written in Python. This tool transforms your terminal into a live dashboard to keep track of your computer's health.

### ✨ Features:
* **Real-Time Tracking:** Monitors CPU, RAM, and Disk space.
* **Network Speeds:** Displays live Upload and Download speeds.
* **Visual Alerts:** Dynamic color-coding (Green/Yellow/Red) changes automatically based on system load.

---
